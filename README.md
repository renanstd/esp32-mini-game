# esp32-mini-game

![demo-gif](images/demonstration.gif)

## Descrição

Neste projeto, vou tentar montar e programar um minigame do zero, usando a placa ESP32, um visor OLED, um potenciômetro e um buzzer.

## Componentes

- Placa ESP32
- Visor OLED
- Potenciômetro 10k
- Buzzer
